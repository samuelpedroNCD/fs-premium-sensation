Política de Privacidade

Última actualização: 28 de Abril de 2026

A Premium Sensation, Lda. (doravante "Premium Sensation", "nós" ou "nosso") compromete-se a proteger a sua privacidade. Esta Política de Privacidade descreve como recolhemos, utilizamos e protegemos as suas informações pessoais quando visita o nosso website ou utiliza os nossos serviços.

1. Responsável pelo Tratamento de Dados

A entidade responsável pelo tratamento dos seus dados pessoais é a Premium Sensation, Lda., com sede em Lisboa, Portugal. Para questões relacionadas com privacidade, pode contactar-nos através de privacidade@premiumsensation.com.

2. Dados que Recolhemos

Podemos recolher as seguintes categorias de dados pessoais:

  • Dados de identificação: nome, endereço de e-mail, número de telefone
  • Dados do evento: tipo de evento, data, local, orçamento estimado e expectativas fornecidas através do formulário de contacto
  • Dados de navegação: endereço IP, tipo de browser, páginas visitadas, duração da visita (recolhidos automaticamente)

3. Finalidade e Base Legal do Tratamento

Tratamos os seus dados para as seguintes finalidades:

  • Resposta a pedidos de consulta — base legal: execução pré-contratual (Art.º 6.º, n.º 1, al. b) do RGPD)
  • Prestação de serviços contratados — base legal: execução do contrato
  • Cumprimento de obrigações legais — base legal: obrigação legal
  • Melhoria do website e análise de desempenho — base legal: interesse legítimo

4. Conservação dos Dados

Os seus dados pessoais são conservados apenas pelo tempo estritamente necessário para cumprir as finalidades para as quais foram recolhidos. Os dados de potenciais clientes que não resultem numa relação contratual são eliminados no prazo de 12 meses. Os dados de clientes são conservados pelo período legal aplicável.

5. Partilha de Dados

Não vendemos nem partilhamos os seus dados pessoais com terceiros para fins comerciais. Os dados podem ser partilhados com fornecedores de serviços que nos auxiliam na prestação dos nossos serviços (ex.: plataformas de e-mail, serviços de alojamento web), sempre sob acordos de processamento de dados que garantem a proteção dos seus dados.

6. Os Seus Direitos

Ao abrigo do RGPD, tem os seguintes direitos:

  • Direito de acesso aos seus dados pessoais
  • Direito de rectificação de dados incorrectos ou incompletos
  • Direito ao apagamento ("direito a ser esquecido")
  • Direito de limitação do tratamento
  • Direito de portabilidade dos dados
  • Direito de oposição ao tratamento

Para exercer qualquer um destes direitos, contacte-nos em privacidade@premiumsensation.com. Tem também o direito de apresentar reclamação à Comissão Nacional de Proteção de Dados (CNPD).

7. Segurança dos Dados

Implementamos medidas técnicas e organizativas adequadas para proteger os seus dados pessoais contra acesso não autorizado, alteração, divulgação ou destruição. No entanto, nenhum sistema de transmissão de dados pela Internet é completamente seguro.

8. Cookies

Utilizamos cookies para melhorar a sua experiência no nosso website. Para mais informações, consulte a nossa Política de Cookies.

9. Alterações a esta Política

Reservamo-nos o direito de actualizar esta Política de Privacidade sempre que necessário. A versão mais recente estará sempre disponível nesta página com a data de última actualização.

10. Contacto

Para qualquer questão sobre esta Política de Privacidade ou sobre o tratamento dos seus dados pessoais, contacte-nos em privacidade@premiumsensation.com.
